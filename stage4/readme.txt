To compile and run the stage 4 project, do the following:

1. Open the project on NetBeans. 
2. Click the "Run" dropdown menu.
3. Click "Set Project Configuration", and click "Customize"
4. Make sure the Main Class attribute is specified as "Main"
5. Return and click the "Run" dropdown menu again
6. Click "clean and build project"
7. Inside the stage 4 folder, there will be a folder called "dist" which contains a jar file.
8. Double click the jar file and the project will run.
9. Another way to run this from terminal is to go to the folder where the jar file is located and write "java -jar movieTheaterOOP.jar"
